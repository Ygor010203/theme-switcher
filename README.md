# Tema Claro/Escuro - Projeto Fundação Bradesco

Site simples com alternância de temas usando CSS Variables e JavaScript vanilla.

## Tecnologias
- HTML5
- CSS3 (variables, flex/grid)
- JavaScript (event listeners, class toggle)

## Como testar
1. Abra `index.html` no browser.
2. Clique no botão pra trocar tema.
